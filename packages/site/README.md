# @otakit/site

OtaKit public website — landing page, docs, contact, and legal pages. Hosted at `otakit.app`.

## Dev

```bash
pnpm install
pnpm --filter @otakit/site dev
```

## What lives here

- Landing page
- Documentation (setup, plugin API, CLI, REST API, guides, security, self-hosting)
- Contact form
- Terms of Service, Privacy Policy
- llms.txt
