# Reporting a Security Vulnerability

Please report security issues privately:

- **Email**: [security@otakit.app](mailto:security@otakit.app)
- **GitHub**: [Private vulnerability reporting](https://github.com/OtaKit/otakit/security/advisories)

Do not open public issues for security vulnerabilities. We aim to respond within 7 business days.

For details on how OtaKit secures update delivery, see [otakit.app/docs/security](https://otakit.app/docs/security).
